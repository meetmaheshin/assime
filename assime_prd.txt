# Project: A.A.T.H - AI Personal Assistant

Version: 1.0

---

# Vision

Build an Android-first AI Personal Assistant that behaves like a real executive assistant instead of a chatbot or to-do app.

The assistant should:

- Remember everything the user has committed to.
- Keep the user accountable.
- Ask follow-up questions.
- Learn from previous conversations.
- Prevent duplicate work.
- Understand projects instead of isolated tasks.
- Be proactive instead of reactive.
- Feel like talking to A.A.T.H.

The assistant should become smarter every day by remembering history.

# AI

OpenAI GPT-5.5 (recommended)

The AI should NOT be called for every reminder.

Use AI only when reasoning is required.

Examples:

✓ Planning

✓ Reflection

✓ Memory search

✓ Breaking tasks

✓ Coaching

✓ Weekly review

Do NOT call AI for:

- simple reminders
- alarms
- notifications
- timers

These should be handled locally.
